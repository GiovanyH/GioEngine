#### Linux build:
clone the repository  
`git clone https://github.com/GiovanyH/GioEngine`  
update submodules  
`git submodule init && git submodule update`  
`mkdir build`  
`cd build/`  
`sh build.sh` or  
`cmake ..`  
`cmake --build .`  
`./GioEngine`
