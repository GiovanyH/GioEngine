#### Current version: 1.2 / Atumalaka

#### Linux build:
clone the repository  
`git clone --recurse-submodules https://github.com/GiovanyH/GioEngine`  
`mkdir build`  
`cd build/`  
`cmake ..`  
`cmake --build .`
